Get的代码管理原理




SVN:集中是代码管理工具

Git：分布式代码管理工具


集中式代码管理工具和分布式代码管理工具区别在于

集中是代码管理工具提交代码的时候很死板，必须提交到服务器上，才会被保存

分布式代码管理工具会在本地计算机上建库，并将代码先保存在本地库中，如果没有网，就先在本地库中储存等网络联通之后，在提交到服务器中，

git命令

1.在本地创建这个小库，

git.init

这条命令执行成功的标志，是所在路径新创建出来一个，名为。get的隐藏文件夹

2.显示git版本管理工具的控制范围

git.status,会对git管理范围内的所有文件进行监听，监听这些文件处于什么状态、、


3如何把本地开发者编写的代码提交到暂存区，使用git   add命令就可以

git add 文件相对路径
将制定文件提交到暂存区
git add .
将当前目录下的所有文件提交到暂存区
git add *
将当前目录下的所有文件提交到暂存区
